命名规则

  • 变量命名：第一个单词小写，第二个单词首字母大写
  • 常量名：所有字母大写，多单词时每个单词用下划线连接
  • 类名，接口名：单词首字母大写。
  • 包名：所有字母都小写。

变量使用及数据类型

  • 变量先声明，后使用。
  • 变量类型
  • 8种数据类型
    1. 整数类型（byte,short,int,long)
    2. 浮点类型（float,double）
    3. 字符型（char）
    4. 布尔型（boolean）

引用数据类型

  • class类
  • interface 接口
  • array接口

常用数据类型及内容

整型
  • byte 1个字节=8bit 范围-128~127
  • short 2字节
  • int 4字节
  • long 8字节 定义long型的时候必须以小写的l或大写L结尾。
浮点型
  • float 单精度4个字节 float类型变量时，变量要以f或F结尾
  • double 双精度8个字节

通常定义浮点型变量时用double类型

字符型

字符型char 1字符=两字节
定义char型变量，通常使用一对’ ‘,内部只能写一个字符

boolean 布尔类型

只有两个值：true 和 flase

基本数据类型运算规则

自动类型提升:
当容量小的数据类型变量和容量大的数据类型的变量做运算时，
结果自动提升为容量大的数据类型
当byte char short三种类型的变量做运算时，结果为int型。

强制类型转换:
需要使用强转符:() 可能导致精度损失

String类型变量的是引用:
String属于引用数据类型 字符串
声明String类型变量时，是引用一堆” “ ；同时String可做连接运算 结果仍然时String类型

进制转换

高位上的0为正 1为负

比如:

1 0 0 0 1 1 1 0 -14原码
1 1 1 1 0 0 0 1 -14反码
1 1 1 1 0 0 1 0 -14补码 反码加1 计算机底层都以补码的方式存储数据

运算符相关知识

  • 比较运算符的返回值都是布尔类型

  • 逻辑运算符中&和&&(或运算同理)

    • 相同点：当符号左边是true时，二者都会执行符号右边的运算
    • 不同点：当符号左边是false时，&继续执行符号右边的运算。&&不再执行符号右边的运算

交换两个变量

异或：相同的变0不同变1

例如：

a 00101 //10
b 01010 //20
c 01111 //30

c = a^b = (c^b)^a

生成随机数

公式：[a,b] : (int)(Math.random() * (b - a + 1) )+ a
